# Lab 1 - Monolithic vs Microservices Architecture

## Overview
This laboratory compares two software architecture approaches:
- **Monolithic** (SAR2) — single unified application
- **Microservices** (student-course-microservices) — independent services

## Folder Structure

├── student-course-microservices/  # Microservices implementation (Python/Flask)
└── Reports/                       # Documentation and reports

## Technologies Used

### Microservices
